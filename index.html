Crystals Collector!

There are 4 crystals displayed as buttons on the page.

Game starts with the display of a Random Number

Goal is to achieve the random number by clicking the crystals in such a way that it matches the random number

Each Crystal is equivalent to a Number that is hidden until the click of the crystal

Score Matches Random Number You Win!!

Score Doesn't match Random Number You Loose!!
